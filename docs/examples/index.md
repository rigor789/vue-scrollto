# Examples

Here are some examples of `vue-scrollto`!

Every example assumes you have installed `vue-scrollto`:
```js
const Vue = require('vue')
const VueScrollTo = require('vue-scrollto')
Vue.use(VueScrollTo)
```

## Using string literals

```html
<button v-scroll-to="'#element'">
    Scroll to #element
</button>

<h1 id="element">Hi. I'm element</h1>
```

## Using object literals

```html
<button v-scroll-to="{ el: '#element' }">
    Scroll to #element
</button>
```
or
```html
<button v-scroll-to="{ element: '#element' }">
    Scroll to #element
</button>
```

### Adding callbacks
```html
<button v-scroll-to="{ 
        el: '#element',
        onDone: onDone,
        onCancel: onCancel
    }">
    Scroll callbacks
</button>
```

```js
export default {
    methods: {
        onDone() {
            alert('done')
        },
        
        onCancel() {
            alert('canceled')
            console.log(e)
        }
    }
}
```
